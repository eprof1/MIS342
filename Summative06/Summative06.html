Learning Objectives

Learning Objectives

  1. Design advanced forms using different form tools and types.
    • Example: An event planner creates a split form to manage event registrations, displaying a list of attendees alongside detailed registration information.
  2. Implement calculated controls and manage tab order within forms.
    • Example: An invoicing form includes calculated controls to automatically compute total amounts and tax, with a logical tab order to streamline data entry.
  3. Use headers, footers, and form controls to enhance form functionality.
    • Example: A non-profit organization customizes forms with headers for branding and controls to toggle between donor information and donation history.
  4. Integrate subforms and control field placement for optimal data entry and display.
    • Example: A construction firm uses subforms to link project details with associated tasks and materials for comprehensive project management.
