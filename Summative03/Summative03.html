Learning Objectives

Learning Objectives

  1. Update data within tables using basic queries.
    • Example: A retailer uses a query to update the price of all products from a specific supplier after a negotiation for a new pricing structure.
  2. Design and execute multi-table queries to extract meaningful information.
    • Example: A travel agency creates a multi-table query to display customer details alongside their booked trips to generate customized itineraries.
  3. Apply sorting and filtering techniques to organize query results.
    • Example: An automotive parts distributor sorts and filters inventory data to quickly find parts below a certain stock level, aiding in reordering decisions.
  4. Use query criteria and multiple query conditions to refine data selection.
    • Example: A financial analyst uses query criteria to filter transactions by date and amount to identify high-value transactions for auditing.
  5. Create calculated fields and use aggregate functions to perform data analysis.
    • Example: A real estate firm calculates the average sale price of homes in different neighborhoods to guide pricing strategies.

Exercise 1-Club Table

You are the Director of Students at Wilfred Academy, a school in New England. Your have just hired a new assistant to help with the clubs at the school. You want to be able to have instant access to the clubs and the faculty advisor for each club. You set up the Clubs.accdb database with two tables, Club and Advisor. There are currently 12 clubs in the table and you need to enter the faculty data. You research the Internet for information on the clubs and work with this database to complete the Club and Advisor tables.

  • Start Access and open the Clubs.accdb database. Save the database as Clubs-TA3.accdb.
  • As the director, you want the database to include one Web site that is a primary source of information for each club. You also want the database to include the name of one magazine that the club should subscribe to.
  • Use your favorite search engine to research Web sites and magazines for each of the 12 clubs.
  • Add a new hyperlink field in the Club table and enter one Web site URL for each club.
  • Add a new text field in the Club table for the magazine name. Enter a magazine for each club. View the table datasheet, and then resize all columns to their best fit.
